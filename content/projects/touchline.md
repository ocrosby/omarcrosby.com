Touchline (working name) is a touch-centric, sport-agnostic match event–tracking web app. Event types aren't hardcoded — they're stored as data rows with behavioral flags, so a new sport is a config change, not a code change.

**Stack:** Next.js App Router · TypeScript · PostgreSQL · Drizzle ORM · Auth.js v5 · Tailwind · Zod · Docker
